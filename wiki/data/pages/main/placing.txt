====== Размещение участников ======

На этой странице содержится обзор процедуры размещения. Работа с формой размещения описана **[[objects:dataprocessors:placingform|здесь]]**.

Всех участников мероприятий можно разделить на тех, кому необходимо размещение, и тех, кому размещение не требуется (например, они живут дома и приезжают только на собрания).

----
===== Данные о размещении в бланке регистрации =====

В бланке регистрации есть блок "Размещение" (см. скриншот ниже). В этом блоке находится флажок "Необходимо", который означает, что участнику необходимо размещение. При создании нового бланка регистрации этот флажок установлен по умолчанию. Если участнику не требуется размещение, то в ходе предварительной регистрации необходимо снять это флажок.

{{ :about:razmeshchenie_blank.jpg | Бланк регистрации }}

Если снять флажок "Необходимо размещение", то все поля (("Объект", "Комната", "Разместить с" и "Комментарий")) в блоке "Размещение" (кроме флажка "Транспорт") будут очищены и станут недоступными для заполнения.

----

===== Данные о размещении в списке регистрации =====

В списке регистрации есть колонка "Р." (Размещение), в которой отражены флажки "Необходимо размещение" из бланков регистрации. Можно сделать отбор списка по реквизиту "Необходимо размещение".

{{ :about:spisok_registr.jpg | Список регистрации }}

----

===== Размещение участников =====

^ {{:about:attention.png |}}Чтобы организовать размещение участников для текущего мероприятия, сначала необходимо указать для этого мероприятия конкретные объекты размещения. ^

Чтобы указать объекты размещения для текущего мероприятия, необходимо открыть карточку мероприятия и на вкладке "Размещение" добавить те объекты размещения,((Гостиницы, общежития, дома святых и др.)) которые будут использоваться. Если объекты размещения в карточке мероприятия не указать, дерево размещения в форме размещения будет незаполненным.

{{ :main:event_placing.jpg | Карточка мероприятия (размещение) }}

Если в карточке мероприятия не указаны объекты размещения, то размещение участников будет невозможным.

Размещение участников можно осуществить двумя способами:

  * через бланк регистрации;
  * через форму размещения.

----
==== Размещение через бланк регистрации ====

Для процедуры размещения в бланке регистрации используется блок размещения, который на скриншоте ниже обозначен рамкой "1".

{{ :about:razmeshchenie_blank2.jpg | Бланк регистрации }}

В поле **"Объект"** выбирается объект размещения.((Гостиницы, общежития, дома святых и др.)) Для заполнения этого поля нужно либо просто начать вводить название объекта размещения (если вы его знаете), либо выбрать из списка объектов, указанных для текущего мероприятия (для этого нужно нажать кнопку выбора - кнопка с многоточием).

В поле **"Комната"** выбирается комната (если у объекта размещения есть комнаты). Для заполнения этого поля нужно либо ввести номер/название комнаты (если вы знаете конкретную комнату), либо выбрать из списка свободных комнат, нажав на кнопку выбора. При нажатии на кнопку выбора появится окно со списком комнат с указанием их категории и количества свободных мест.

В поле **"Разместить с"** можно выбрать соседей - участников, с которыми необходимо разместить текущего участника.

Поле **"Комментарий"** предназначено для комментариев к размещению.((На скриншоте приведён пример комментария.)) Комментарий размещения будет выводиться в форме размещения и его можно заполнить заранее, например, во время предварительной регистрации.

^ {{:about:attention.png |}}Комментарии размещения нужно записывать именно в это поле, а не в поле для общих комментариев, которое расположено в нижней части бланка регистрации. ^

В блоке размещения также есть флажок **"Координатор"**. Если установить этот флажок, текущий участник будет отмечен координатором комнаты. Если такой флажок был установлен у другого участника, то он автоматически будет снят, поскольку в комнате может быть только один координатор.

  До того, как для текущего мероприятия указаны объекты размещения, в блоке
  размещения доступны для заполнения все поля, кроме полей "Объект" и "Комната".

Рамкой "2" на скриншоте обозначен сводный список совместного размещения. Этот список формируется системой следующим образом:

  * учитываются участники, которые указаны в поле "Разместить с" бланка регистрации текущего участника;
  * учитываются участники, у которых в бланке регистрации текущий участник указан в поле "Разместить с".

Если сводный список совместного размещения пуст, то блок, обозначенный рамкой "2", не отображается в бланке регистрации.

----
==== Размещение через форму размещения ====

[[objects:dataprocessors:placingform|Форма размещения]] предназначена для распределения участников по комнатам. Форму размещения можно открыть с рабочего стола базы регистрации. В разделе "Последние мероприятия" необходимо выделить строку мероприятия, по которому будет проводиться размещение, и нажать кнопку с изображением домика (см. скриншот).

{{ :about:forma_razm_otkr.jpg | Открытие формы размещения }}

Работать с формой размещения в режиме редактирования может только один пользователь базы регистрации. Для всех остальных пользователей в это время форма может быть открыта только в режиме просмотра.

Работа с формой размещения описана **[[objects:dataprocessors:placingform|здесь]]**.
