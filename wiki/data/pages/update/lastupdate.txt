===== Новое в версии 3.3 (июль 2013) =====

----

**[[update:3_3#Обновление алгоритма поиска участника в картотеке|Обновление алгоритма поиска участника в картотеке]]**

В версии 3.3 обновлён алгоритм поиска участника в картотеке (через поле поиска на рабочем столе). Поиск участников стал более результативным.

При поиске участников необходимо понимать следующие правила: 1) поиск учитывает язык ввода, 2) при поиске по ФИО система учитывает только первые два слова, 3) при поиске по ФИО можно задавать условие поиска, 4) можно искать участника по номеру телефона.

===== Новое в версии 3.2 (июль 2013) =====

----

**[[update:3_2#Модуль онлайн-регистрации|Модуль онлайн-регистрации]]**

Модуль онлайн-регистрации запущен в режиме бета-тестирования...

По ссылкам, приведённым ниже, можно посмотреть видеоинструкции о работе с сайтом для онлайн-регистрации.

[[http://youtu.be/_JbNcwoQ8RQ|Регистрация на сайте reg-page.ru]] (продолжительность - 8:41). Это видео для новичков, которые будут самостоятельно регистрироваться по приглашению, а также для святых из тех городов, где нет ответственных за регистрацию. В нём содержится обзор страницы посетителя сайта и описан процесс самостоятельной регистрации.

[[http://youtu.be/eO9kO7JBDCk|Обзор страницы администратора сайта]] (продолжительность - 4:34). В этом видеоролике представлен сайт регистрации и сделан обзор страницы администратора сайта. Видео предназначено для святых, которые отвечают за регистрацию в церквях или регионах.

[[http://youtu.be/7GSbNubyZcA|Обзор страницы администратора сайта]] (продолжительность - 7:58). В этом видео описан процесс регистрации святых на конференции и обучения через страницу администратора сайта.

===== Новое в версии 3.1 (июль 2013) =====

----

**[[update:3_1#SMS-уведомления участникам мероприятий|SMS-уведомления участникам мероприятий]]**

Основное новшество версии 3.1 - возможность отправки персональных и массовых sms-уведомлений участникам мероприятий. Таким образом, с третьей версии программы мы переходим на новый уровень работы, подразумевающий более тесное взаимодействие между командой регистрации и участниками мероприятий (или ответственными за регистрацию в регионах).

Краткий видеообзор нового инструмента можно посмотреть [[http://youtu.be/_9-cIXQrJ1g|в этом видео]]. Более детальный обзор смотрите [[http://youtu.be/q0pXTlEukNk|по этой ссылке]].

===== Новое в версии 2.35 (июнь 2013) =====

----

**[[update:2_35#Добавлен быстрый поиск в списке регистрации|Добавлен быстрый поиск в списке регистрации]]**

Теперь можно искать необходимый бланк в списке регистрации вводом фамилии в поле быстрого поиска, а также вводном номера регистрации. См. [[http://youtu.be/9CmMuTGbKuY|видео]]

**[[update:2_35#Изменена форма бланка регистрации|Изменена форма бланка регистрации]]**

  * Добавлена фотография участника;
  * На первую вкладку перенесены данные о регистрационном взносе;
  * Добавлен блок с информацией о датах создания и изменения, а также авторе последних изменений бланка регистрации.

См. [[http://youtu.be/R3fF-x2zA0k|видео]].

===== Новое в версии 2.34 (май 2013) =====

----

**[[update:2_34#Изменена карточка мероприятия|Изменена карточка мероприятия]]**

Изменения связаны, главным образом, с предстоящим переходом к онлайн-регистрации. См. [[http://youtu.be/vdK95l--pLI|видео]]

**[[update:2_34#Доработан поиск участника в картотеке|Доработан поиск участника в картотеке]]**

Теперь можно искать участника не только по русским фамилиям и именам, но также по английским. См. [[http://youtu.be/R3fF-x2zA0k|видео]].

**[[update:2_34#Добавлен учёт гражданства участников|Добавлен учёт гражданства участников]]**

Данные о гражданстве участников учитываются в том же регистре, что и паспортные данные. См. [[http://youtu.be/dXRyF0ie3T8|видео]]

**[[update:2_34#Добавлены списки паспортных данных и иностранных участников|Добавлены списки паспортных данных и иностранных участников]]**

См. меню "Списки" в форме "Последние мероприятия" на рабочем столе программы.

**[[update:2_34#Добавлена статистика по датам приезда/отъезда|Добавлена статистика по датам приезда/отъезда]]**

Статистика позволяет определить количество человек по датам приезда и отъезда.

**[[update:2_34#Добавлен учёт взносов (в простом виде)|Добавлен учёт взносов (в простом виде)]]**

Система пока выдаёт только общую сумму взносов в разрезе валют. Доработка будет проводиться позже.
