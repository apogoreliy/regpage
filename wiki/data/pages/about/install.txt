====== Установка тонкого клиента ======

Работа с базой данных, размещённой на веб-сервере в Интернете, осуществляется через [[about:thinclient|тонкий клиент]] - программу, выполняющую функции браузера базы данных. Тонкий клиент работает в операционной системе Windows (XP или 7).

Для установки программы на локальный компьютер выполните следующие действия:

  - Скачайте последнюю версию тонкого клиента [[https://www.box.com/s/hmicn34lxdopep7o0rvd|по этой ссылке]] (архив около 33 МБ);
  - Распакуйте архив и запустите файл **setup.exe**;
  - Пройдите несложные этапы установки программы.

После установки программы на рабочем столе вашего компьютера появится ярлык программы 1С. Для удобства запуска программы выполните следующие действия:

  - Правой кнопкой мыши кликните по ярлыку 1C и выберите пункт "Свойства";
  - В окне свойств перейдите на вкладку "Ярлык";
  - В поле "Объект" к тому, что там написано, __через пробел__ добавьте следующую строку (вместо ЛОГИН и ПАРОЛЬ необходимо подставить данные канала, который предоставлен вам для доступа к базе регистрации; __не удаляйте кавычки__): 

  ENTERPRISE /WS "https://web1.ofis24.net/app/reg/Regteam" /WSN "ЛОГИН" /WSP "ПАРОЛЬ" /L "ru"
  
Строка выше содержит параметры вашего канала доступа к веб-серверу. Её можно называть строкой доступа.  
  
Теперь программу можно запускать:

  - Дважды кликните по ярлыку 1С на рабочем столе;
  - В появившемся окне выберите себя в поле "Пользователь" (выпадающий список);
  - В поле пароль введите ваш пароль для входа в базу регистрации и нажмите кнопку "ОК".

^ {{:about:attention.png |}}Обратите внимание: пароль для входа в базу регистрации - это не то же самое, что пароль для доступа к веб-серверу, который был указан в строке доступа. ^

===== Советы по работе с тонким клиентом =====

Не предоставляйте свои данные для входа в программу((Логин и пароль)) другим. У каждого пользователя программы (даже у того, кто предполагает работать в ней временно) должны быть свои данные для входа.

Если вы планируете сделать перерыв в работе, лучше закрыть программу.

Если тонкий клиент завис или выдал ошибку, проверьте соединение с Интернетом. При необходимости перезагрузите программу. Даже во время сбоя тонкого клиента все данные сохраняются на сервере в Интернете.
